# PowerPulse

PowerPulse is a fitness app for transforming your body shape and lifestyle.

## About project

With the help of this project, people can keep track of their weight, engage in
sports, and lead a healthy lifestyle.

## Technologies used:

- HTML/CSS
- Framer Motion
- JavaScript
- React
- Redux Toolkit
- Pagination
